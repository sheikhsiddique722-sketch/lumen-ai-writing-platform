# ✍️ Lumen AI Writing Platform

A modern AI-powered writing platform concept built as a frontend practice project. Designed with a clean SaaS-inspired interface, the project focuses on creating an intuitive writing experience through elegant UI, responsive layouts, and a professional user experience.

This project was developed to strengthen frontend development skills while recreating the look and feel of a modern AI productivity application.

---

**Website:** https://sheikhsiddique722-sketch.github.io/lumen-ai-writing-platform/

## ✨ Features

* Modern AI SaaS landing page
* Clean and minimal interface
* Responsive design for desktop, tablet, and mobile
* Professional typography and visual hierarchy
* Smooth scrolling navigation
* Interactive call-to-action sections

* User-focused UI/UX design
