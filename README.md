# Lumen — AI Writing Companion

Lumen is a futuristic AI writing companion platform designed to help users write smarter, faster, and more creatively.

Built using HTML, CSS, JavaScript, and Node.js, the project features a modern responsive interface, smooth animations, premium UI design, and AI-inspired productivity tools.

## ✨ Features

- Futuristic Modern UI
- Responsive Design
- AI Writing Dashboard
- Smooth Animations
- Dark Premium Theme
- Smart Productivity Layout
- Interactive Components
- Fast & Lightweight Structure

## 🚀 Installation

1. Clone the repository

git clone https://github.com/your-username/lumen-ai-writing-platform.git

2. Open project folder

cd lumen-ai-writing-platform

3. Install dependencies

npm install

4. Start server

node server.js
